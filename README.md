# KBOOMDEV — Sitio Web Oficial

Sitio web oficial de **KBOOMDEV**, enfocado en ofrecer:

- Desarrollo de aplicaciones web
- Desarrollo de sistemas web a medida
- Landing pages profesionales
- Soluciones digitales para negocios

El objetivo del sitio es presentar los servicios, mostrar trabajos y permitir que potenciales clientes **agenden reuniones directamente desde la web**.

---

# Tecnologías utilizadas

Este proyecto está construido utilizando tecnologías simples, rápidas y modernas:

- HTML5
- Tailwind CSS
- JavaScript
- Calendly (para agendamiento automático de reuniones)

No se utiliza framework frontend pesado para mantener el sitio **rápido y ligero**.

# Integración con Calendly

La página de contacto incluye un sistema de agendamiento automático mediante Calendly.

Esto permite que un cliente pueda reservar un horario disponible directamente desde la web.

Widget utilizado:


https://calendly.com/rubenlopezrolando


Ejemplo de integración:

```html
<div
class="calendly-inline-widget h-[700px] w-full"
data-url="https://calendly.com/rubenlopezrolando?locale=es">
</div>

<script src="https://assets.calendly.com/assets/external/widget.js" async></script>

Esto muestra el calendario directamente en la página.
